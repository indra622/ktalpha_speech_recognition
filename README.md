# ktalpha_speech_recognition
2021년 11월 18일부터 12월 7일까지 했던 숫자가 포함된 음성인식 모델 개발 정리 




# 개요

총 참가 팀: 39팀

데이터: 대략 470시간 분량의 한국어 데이터. 낭독체

실제 참여 기간: 11월 25일 ~ 12월 7일

사용 tool: kaldi, sgspeech_v2 (kaldi는 시간안에 학습 완료 안됨)
